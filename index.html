About Me

Program delivery is
my discipline

I am a Senior Technical Project Manager and Certified SAFe® 6.0 Agilist with over 15 years of structured program delivery experience. I manage complex, multi-team software programs end-to-end — from requirements baseline through production release — coordinating UI, backend, middleware, data, and test automation teams simultaneously.

My strength is bridging the gap between engineering teams and business stakeholders. I enforce phase-gate quality standards, manage program budgets using Earned Value Management, own Change Request governance, and report milestone health to senior leadership and customers. I have successfully delivered B2E and B2C applications, data governance platforms, API/microservices programs, and AI-powered internal tools.

What sets me apart is AI-augmented delivery. I actively use Gemini, NotebookLM, and ChatGPT to accelerate ceremony preparation, decode technical blockers faster, and improve stakeholder communication quality — bringing a modern, forward-thinking approach to every program I lead.

Technical Architecture
Pattern
Serverless Real-time Architecture — frontend communicates directly with Firebase services, no custom backend API needed
Sync
WebSocket/WSS via Firebase RTDB — changes broadcast instantly to all connected users using onValue() listeners
Auth
Anonymous authentication preventing unauthorised writes without requiring user sign-up
Data
Atomic transactions ensuring vote counts are always accurate under concurrent simultaneous voting
CI/CD
GitHub Actions pipeline — CI validates feature branches, CD deploys to production only when CI passes
Security
Firebase security rules, CSP headers, input sanitisation, vote rate limits, HTTPS enforced
